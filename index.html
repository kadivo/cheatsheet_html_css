SVG

Inline SVG tähendab, et SVG (vektorgraafika) kood kirjutatakse otse HTML-i dokumendi sisse, mitte ei kasutata seda eraldi failina või pildina.

Võrdlus teiste viisidega:

  • <img src="icon.svg" /> → SVG käitub nagu tavaline pilt, ei saa elemente seest muuta.
  • background-image: url("icon.svg") → SVG pannakse taustaks, aga seestpoolt ei ole muudetav.
  • Inline SVG → saad iga kujundi (<path>, <circle>, <rect>) eraldi stiliseerida ja animatsioone teha.

    • Inline SVG-s <path> element on kõige paindlikum viis kujundite joonistamiseks. Kuju määratakse d atribuudi abil.

    Olulisemad käsud:

    • M - move to → viib pliiatsi kindlasse punkti (ilma joont tõmbamata).
    • L - line to → tõmbab sirgjoone antud punkti.
    • H - horizontal line to
    • V - vertical line to
    • C - cubic bezier curve (kolme kontrollpunktiga kõver).
    • Q - quadratic bezier curve (ühe kontrollpunktiga kõver).
    • Z või z - sulgeb tee (joonistab viimase punkti ja alguspunkti vahele joone).

    CSS

    CSS ehk Cascading Style Sheets on stiililehtede keel, mida kasutatakse HTML-i või XML-i dokumentide välimuse kujundamiseks. See ei muuda sisu iseennast, vaid määrab kuidas sisu välja näeb, näiteks värvid, fondid, paigutus ja animatsioonid.

    Flexbox

    Flexbox (ehk Flexible Box Layout) on CSS-i paigutusmudel, mis võimaldab elemente järjestada ja paigutada paindlikult ühes mõõtmes - kas reas (horisontaalselt) või veerus (vertikaalselt). See on eriti kasulik siis, kui tahad elemente ühtlaselt jaotada, keskele sättida või muuta nende suurust vastavalt saadaval olevale ruumile.

    Grid

    CSS Grid on paigutussüsteem, mis võimaldab sul veebilehel elemente paigutada kahemõõtmelises võrgustikus - nii ridade kui ka veergude kaupa. See teeb keerulisemate disainide loomise lihtsamaks ja paindlikumaks kui näiteks flexbox, mis töötab põhiliselt ühes mõõtmes (kas rida või veerg).

    Positsioning

    CSS-is position omadus määrab, kuidas element paigutub lehel ja kuidas ta suhestub teiste elementidega. Selle abil saab elemente liigutada täpsemini kui tavalise voo (normal flow) puhul.

    Kui elemendid kattuvad, määrab z-index nende kihituse. Mida suurem z-index väärtus, seda kõrgemal kihil element asub.

    Sticky - Kombinatsioon relative ja fixed positsioonidest.
    Absolute - Elemendi positsioon, lähtudes lähimast positsioneeritud vanemast.
    Static - Kui väärtust pole määratud, on static vaikimisi väärtus. Tavapärane dokumendi voolu järgimine.
    Relative - Elemendi positsioon lähtudes tavalisest asukohast.
    Fixed - Elemendi positsioon on fikseeritud ja ei muutu kerimise ajal.

    Lisainfo: W3S

    Värvid

    HTML ja CSS toetavad palju värve nimede järgi, kuid saab kasutada veel:
    • Hexadecimal colors - #RRGGBB #ff0000 (punane)
    • Hexadecimal colors with transparency - #RRGGBBtransparency-value #ff000050 (punane 50% läbipaistvusega)
    • RGB colors - rgb(red, green, blue) rgb(0 255 0) (roheline)
    • RGBA colors with transparency - rgba(red, green, blue, alpha) rgba(0, 255, 0, 0.5) (roheline 50% läbipaistvusega) NB! kindlasti komad vahele
    • HSL colors - hsl(hue, saturation, lightness) hsl(240, 100%, 50%) (sinine)
    • HSLA colors - hsla(hue, saturation, lightness, alpha), hsla(240, 100%, 50%, 0.5) (sinine 50% läbipaistvusega)
    • currentcolor keyword - kasutab sama värvi, mis divis juba määratud (color: blueviolet; border: 5px solid currentColor)
